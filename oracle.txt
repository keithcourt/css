你连接别人的Oracle，首先你的电脑需要安装Oracle的客户端或者服务端，PL/SQL DEV这只是一个工具而已，没有前面的客户端或服务器你无法连接远程的Oracle，不过由你的描述，我知道你装的是服务端
连接别人的Oracle，你的电脑不需要进行监听配置，配置监听是让别人连你的Oracle，你连他的Oracle，需要他配置监听
假设他没配置监听，他自己也是可以连上自己的Oracle的，所以第一件事是确认他正确配置了Oracle监听，走的是本地服务，不是监听，你的PL/SQL DEV连不上同理
你需要知道他配置监听的服务名，这个在lsnrctl status 命令下可以看到（他电脑上操作）
假设服务名是HUAWEI，你需要在你的电脑上进行连接配置，一种方法是使用Oracle自带的Net Manager工具建立 （Net Manager--》服务命名--->增加/输入IP和之前的监听服务名HUAWEI），另一种方法是在TNSNAME.ORA里面添加一行记录，语法如下：
HUAWEI =
  (DESCRIPTION =
    (ADDRESS = (PROTOCOL = TCP)(HOST = 他电脑IP)(PORT = 1521))
    (CONNECT_DATA =
      (SERVER = DEDICATED)
      (SERVICE_NAME = HUAWEI)
    )
)
  在cmd输入TNSPING HUAWEI 看是否通，如果通的话进行下一步，不通说明第5步有问题
在cmd输入sqlplus ZH/密码@HUAWEI或者在PL/SQL DEV登录
TNSNAME.ORA文件在哪？在Oracle安装的主目录/network/admin/ 下面
还有sqlnet.ora和listener.ora文件
